🤖 AI 金鑰（存在本機瀏覽器，不會上傳）

🏫 校務系統連接

會讀取同網域「Young校務管理系統」存在此瀏覽器的資料（教職員、行事曆、研習、年級班級），讓 AI 的建議貼合你學校實況。注意：開啟後，這些資料會隨你的問題送到你設定的 AI 服務；且需在同一個瀏覽器使用過校務系統才有資料。

📂 會議紀錄資料夾

貼上 Google Drive 資料夾網址，在會議功能顯示快速開啟按鈕。

📅 Google 行事曆

貼上 Google 日曆的 ICS 私人網址，自動顯示今明行程。
取得方式：Google 日曆 → 設定 → 你的日曆 → 整合日曆 → 私人網址（iCal）

📲 LINE 推播

設定後，AI 產出結果可一鍵傳到您的 LINE。
User ID：傳訊息給 Bot 後，Bot 會回覆您的 ID。

🚀 GitHub 直接部署（Abot）

貼上 GitHub Personal Access Token，讓「App／系統開發」情境產出的完整程式碼可以在對話裡按一下直接部署到 cyysongyy 帳號的 GitHub Pages。
Token 只存在此瀏覽器，直接從瀏覽器呼叫 GitHub API，不會經過任何第三方伺服器——跟你貼的 AI 金鑰是同樣的信任模型。建議在 GitHub 產生 token 時將權限限縮為只有你要用的幾個 repo，並定期更換。
